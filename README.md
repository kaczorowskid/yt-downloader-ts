# YT-Downloader

![yt-downloader](https://user-images.githubusercontent.com/57078565/141660604-9fb41a17-7448-4f29-927c-5c292466dc19.jpg)

An application that allows to download music from YouTube.

In project used:
- typescript
- react
- ytdl-core
- express.js
- sequelize
- pusher
- jsonwebtoken
- bcypt
- nodemailer
